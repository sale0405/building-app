# Business Help (frontend)

Request help, post offers, match, schedule meetings.

## Dependencies

- `core/api-client`

## Routes

| Path | Page | Nav |
|------|------|-----|
| `/business-help` | BusinessHelpPage | Business Help |

## API used

`/business-help/requests`, `/offers`, `/matches`, `/matches/:id/meetings`

## Events

Match triggers chat + notifications server-side.
