# Food Locker (frontend)

Browse and manage food locker listings.

## Dependencies

- `core/api-client` — multipart upload for photos

## Routes

| Path | Page | Nav |
|------|------|-----|
| `/food-locker` | FoodLockerPage | Food Locker |

## API used

`/food-locker/listings` — list, create, reserve, sold, rate

## Events

None on frontend; seller notified via notifications module.
